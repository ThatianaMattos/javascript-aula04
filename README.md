# javascript-aula04

JavaScript Aula 04

Arrays, Métodos de Array e Exercícios Práticos

Este repositório contém o conteúdo da Aula 04 do curso, incluindo explicações, exemplos, exercícios e um mini projeto prático usando Arrays e seus principais métodos.

A estrutura inclui a aula principal e uma pasta separada para exercícios, permitindo organização e evolução do aprendizado.

Conteúdo da Aula

Nesta aula trabalhamos:

Introdução a Arrays

O que é um array

Como acessar itens

Como modificar valores

Como adicionar e remover elementos

Métodos de Array

Métodos estudados:

push()

pop()

unshift()

shift()

splice()

slice()

reverse()

sort()

includes()

indexOf()

Cada método foi aplicado em exemplos reais e comentados no arquivo script.js.

Mini Projeto

Dentro da pasta aula04, você encontrará:

index.html

script.js

Demonstração dos métodos

Projeto de Lista de Tarefas com manipulação de array

Exibição no HTML do resultado final

O projeto aplica:

adicionar tarefas

remover

substituir

ordenar

fatiar tarefas usando slice

exibir resultados no console e no HTML
